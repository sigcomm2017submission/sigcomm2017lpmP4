# Sigcomm 2017 LPM (P4 part)

## Prerequisites

 1. [CMake](https://cmake.org/)
 2. [Boost](http://www.boost.org/)
 3. [Python2.7](https://www.python.org/)

## Compilation

 1. Clone the repository.
 2. __Change__ to the repository root.
 3. Change to the optimization algorithms root:

     ```bash
     cd p4t_native
     mkdir build
     cd build
     cmake ..
     make
     ```
